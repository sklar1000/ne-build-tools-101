# AI Build Tools 101 — For MBA Students Who've Never Coded

You're about to build something real. Not a slide deck. Not a memo. A working tool that a stranger can use — a quiz, an assessment, a landing page with a checkout button, a personalized recommendation engine.

You don't need to know how to code. You need to know how to describe what you want.

## What You'll Build

You have a riskiest assumption and a Value Staircase from class. Now you'll build artifacts at each level:

| Level | What You Build | Example |
|-------|---------------|---------|
| **L1: Information** | A guide, checklist, or resource your customer would seek out | "Complete Guide to Raw Dog Food for Houston Pet Owners" |
| **L2: Assessment** | A quiz, calculator, or audit that diagnoses THEIR specific problem | "Is Your Dog Getting the Right Nutrition?" — 8 questions, personalized report |
| **L3: Partial Solution** | Something that actually solves part of their problem | Custom 4-week meal plan generated for their specific dog |
| **L4: Transaction** | A way for them to pay you or commit | $29 deposit page for first month's delivery box |

**With AI tools, L1 takes 30 minutes. L2 takes 2-4 hours. These used to take weeks.**
